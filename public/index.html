Image Search Abstraction Layer

The service query for images and shows the most recently submitted search strings

Usage Examples:

  • https://imagesearch-fcc-api.herokuapp.com/query/lolcats?offset=2

Usage Output:

  • [{"url":"http://data.whicdn.com/images/30855761/large.jpg","alt":"cat and funny image","pageurl":"https://encrypted-tbn2.gstatic.com"}, ...]

Usage Examples:

  • https://imagesearch-fcc-api.herokuapp.com/latest/

Usage Output:

  • [{"term":"lolcats funny","when":"2016-08-11T20:07:52.175Z"},...]
